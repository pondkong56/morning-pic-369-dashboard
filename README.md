# Morning Pic 369 Dashboard

แดชบอร์ดสำหรับคุมงานขาย แชท แอด ออเดอร์ งานผลิต และรายงานของธุรกิจ Morning Pic 369

## ไฟล์หลัก
- `index.html` — หน้า showcase สำหรับพรีเซนต์ภาพรวมระบบ
- `app.html` — หน้าแอปต้นแบบแบบ local-first
- `SPEC.md` — ขอบเขตระบบและข้อมูลที่ต้องมี
- `TASKS.md` — รายการงานพัฒนาตามลำดับ
- `architecture.md` — สถาปัตยกรรมชั้นถัดไปสำหรับ AI agent / backend sync / database กลาง
- `sync-contract.md` — โครงสัญญาการ sync ระหว่างหน้าแอปกับ backend
- `db-schema.sql` — โครงฐานข้อมูลกลางเบื้องต้น
- `api-contract.md` — สัญญา API สำหรับ backend worker

## วิธีเปิดดู
เปิด `index.html` หรือ `app.html` ในเบราว์เซอร์ได้โดยตรง

## หมายเหตุ
เวอร์ชันนี้ยังเป็นต้นแบบที่เน้น local-first และเปิดทางไว้สำหรับ AI agent, backend sync และฐานข้อมูลกลางในเฟสถัดไป
